About This Project

AI BrainCare is a deep learning based web application developed as an MCA Mini Project under Artificial Intelligence & Data Science specialization.

The system uses Transfer Learning with MobileNetV2 architecture to classify brain MRI images into four categories: Glioma, Meningioma, Pituitary Tumor, and No Tumor.

This project demonstrates the practical application of Artificial Intelligence in healthcare imaging analysis while maintaining ethical responsibility and medical disclaimer standards.

Technologies Used

  • Python
  • TensorFlow & Keras
  • Transfer Learning (MobileNetV2)
  • Flask Web Framework
  • HTML, CSS
  • Image Data Augmentation
